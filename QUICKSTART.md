# 快速开始指南

## 🎯 项目已准备就绪！

### 方式一：使用启动脚本（推荐）

```bash
./start.sh
```

### 方式二：手动启动

```bash
# 1. 激活虚拟环境
source venv/bin/activate

# 2. 运行服务器
python app.py
```

## 🌐 访问地址

服务器启动后，在浏览器中访问：

- **登录页面**: http://localhost:8080/login.html
- **注册页面**: http://localhost:8080/register.html
- **仪表盘**: http://localhost:8080/dashboard

## ✨ 功能演示

### 1. 登录流程
1. 打开 http://localhost:8080/login.html
2. 输入用户名：`admin`
3. 输入密码：`admin`
4. 点击"登录"按钮
5. 成功跳转到仪表盘

### 2. 注册新用户
1. 点击登录页面的"立即注册"
2. 填写用户名（至少 3 个字符）
3. 填写邮箱（可选）
4. 设置密码（至少 6 个字符）
5. 确认密码
6. 勾选同意服务条款
7. 点击"注册"
8. 注册成功后自动跳转回登录页

### 3. 记住我功能
- 登录时勾选"记住我"
- 下次访问会自动填充用户名

## 🛑 停止服务器

按 `Ctrl + C` 组合键

## 📁 重要文件说明

- `app.py` - Flask 后端主程序
- `users.json` - 用户数据存储（会自动创建）
- `templates/` - HTML 模板目录
- `static/css/style.css` - 样式文件
- `static/js/` - JavaScript 文件

## 🔧 常见问题

### 端口被占用？
修改 `app.py` 中的端口号：
```python
app.run(debug=True, host='0.0.0.0', port=8080)
```

### 无法启动？
确保已安装 Python 3，并检查虚拟环境是否正确激活。

### 忘记密码？
可以直接编辑 `users.json` 文件重置密码（密码为 SHA256 加密）。
